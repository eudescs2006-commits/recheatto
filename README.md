# Recheatto

![Logo](frontend/public/logo.png)

**Sistema completo de delivery artesanal com identidade moderna — pedidos, pagamento e rastreio em tempo real.**

## Estrutura do repositório

```
recheatto/
├── backend/        ← API Node.js com Express + integração de pagamentos
├── frontend/       ← React + Vite (cardápio, checkout, painel admin)
├── .env.example    ← modelo de variáveis (backend/.env.example)
├── README.md       ← esta documentação
└── deploy-guide.md ← guia passo a passo (Vercel + Railway)
```

## Quick start

Veja as instruções completas em `deploy-guide.md` e nos READMEs dentro de `backend/` e `frontend/`.
